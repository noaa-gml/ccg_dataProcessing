Printing cylinder label instructions

  1. Locate a computer with a Brother label printer, P-Touch software, and network access. Make sure that the label printer is turned on.
  2. Login to the computer if necessary.
  3. Browse to image directory. There may be a ‘shortcut to labels’ shortcut. Otherwise, use Start Menu -> Run -> \\ccgg\ccg\refgas\label\labels\ -> OK
  4. Select the image file(s) to print. Use the ‘Shift’ key to select a range. Use the ‘Ctrl’ to select multiple images. Right click on the images and select ‘Print’. A print dialog box should appear.
  5. Print dialog first page is ‘Printing Options’.
    1. Select the Brother printer (e.g., Brother PT-9600).
    2. Select ‘Printing Preferences’. Format should be ‘Cylinder label’ ( tape: ½”,length: 2.25”, feed: 0.08”, orientation: landscape). Click ‘OK’.
    3. Click ‘Next’.
  6. Print dialog second page is ‘Layout Options’.
    1. Select ‘Full page photo print’.
    2. Make sure that the label in the preview is correct. No text should be cut off. The image size and text looks normal. If there is a problem, please click ‘Back’ and re-do step 5.
    3. Click ‘Next’ when ready.
  7. The label(s) should now print.
  8. Once printing is complete, close all related dialog boxes.
  9. Return computer to state you found it. This may mean logging off the computer and turning off the label printer.

Resetting mobile device password

  1. Log into Refgas Manager on a mobile device using your Windows/Linux password.
  2. From the Home screen, click 'Account'.
  3. Type your Windows/Linux password in the current password.
  4. Set your new password. Use the scanner if desired.
  5. Verify your new password. Use the scanner if desired.
  6. Click 'Update' to complete.

Mobile device can't connect to network

  1. Restart the Wi-Fi
    1. Click the Home icon at the bottom of the screen. It looks like a little house.
    2. On the device desktop on the left tap on the Wi-Fi icon. It should turn from blue (ON) to gray (OFF).
    3. Tap the Wi-Fi icon again to turn it back on.
    4. Try to connect to Refgas Manager again.
