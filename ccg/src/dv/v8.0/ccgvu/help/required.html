Requirements for CCGVU

The ccgvu programs is comprised of a graphical interface for the user, and a backend program for the reading data and filtering the input. The graphical interface is written in tcl/tk, and the backend programs are written in C. Requirements for compiling and running ccgvu are:
  • tcl/tk version 8.3 or later.
    Available at http://www.scriptics.com
  • The blt tcl/tk widget extension.
    Available at http://sourcforge.net/projects/blt
  • The BWidget widget library for tcl/tk.
    NOTE Only version 1.2 will work. Do not use the later 1.3 version.
    Available at http://sourcforge.net/projects/tcllib
  • The tk html widget.
    This is only needed if you want to use the internal browser for the help pages. Available at http://www.hwaci.com/sw/tkhtml/index.html
  • Beginning with version 5.3.0, the mysqltcl package is needed, for accessign the MySQL database. Available at http://www.xdobry.de/mysqltcl
  • An ANSI C compiler.
