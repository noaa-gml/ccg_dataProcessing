Export [File Menu]

Exports (writes) data to a file consisting of columns of numbers, separated by blanks. These columns are a date, and results from the curve fits to the data.

The following are the options of the Export dialog box:
  • Choose Data Sets To Export
    Select which data sets from the various graphs are to be exported. Some of the data sets are not available if the data is to be exported at equally spaced intervals.

  • Choose Output Options
    Specify whether the data is to exported at equally spaced intervals, or at the dates and times of the original input data points.

    When the 'Dates at Equally spaced intervals' toggle button is selected, the 3 text entry boxes for the starting date are active. The default is the date of the first data point. You can specify a different starting date for exported data by entering new values for the year, month and day. All data points are set to the middle of the day. The intervals are determined by the sampling interval.

    The user can specify whether the dates should be in decimal year format or in a calendar format.

    The user also specifies the output file name.

Action Buttons

  • OK
    Writes to choosen results to the selected file. If the file exists, the user is prompted whether to overwrite the file or not.
  • Close
    Close: Dismisses the dialog box and takes no action.
