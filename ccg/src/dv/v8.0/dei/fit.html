The files with the _fits part of the file name contain the results of the smooth curve fits to the data. There are six columns of data:

  1. The smooth curve fit to the data, with gaps filled in.
  2. The smooth curve fit to the data. Gaps are not filled.
  3. The trend curve fit to the data.
  4. The harmonic part of the function fit to the data.
  5. The smooth curve fit (column 1) minus the trend curve fit, i.e the detrended data.
  6. The number of data points within the mbl time interval.
